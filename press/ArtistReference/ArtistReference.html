Artist Reference & Value Study

brings professional studio tools to traditional artists everywhere.

Download on the App Store

Artist Reference helps traditional artists see their subject clearly and translate what they see onto canvas with confidence. Designed specifically for painters and illustrators working in physical mediums, the app brings together essential reference tools in a focused, elegant interface that supports the creative process without distracting from it.

At its core, Artist Reference makes observation simpler and more intentional. Customizable gridlines help establish accurate proportions from the first mark. Value Studies reduce any image to its underlying light and shadow structure, allowing artists to understand form before mixing a single color. With these tools working seamlessly together, artists can move from reference to canvas with clarity and control.

Inspiration is built in. Curated Collections span Animals, Architecture, Botanical subjects, Master Studies, and more. Each thoughtfully selected and ready to paint. Instead of spending time searching and organizing images, artists can begin working immediately, with every reference stored in one organized, distraction-free space.

The latest update introduces Color Harmonies, an interactive color wheel designed to help artists plan and refine their palettes. Explore triadic, analogous, split complementary, monochromatic relationships, and more. Selected swatches can be saved directly to a secondary reference panel, keeping color decisions visible and accessible throughout the painting process.

"Artist Reference began as a solution to my own frustrations while painting.

I found myself constantly switching between apps for reference images, value studies, and color harmonies; then manually organizing screenshots in Apples Freeform app.

The tools were an interruption. Artist Reference brings those essential tools into one focused space, so the process feels seamless and creativity can take the lead."

Gil Osborne, Indie Developer, Artist Reference

Artist Reference is intentionally simple. There are no ads, no accounts, and no unnecessary complexity. The app works entirely offline, making it perfect for plein air painting, studio sessions, or workshops anywhere. All references live in one organized space, ready the moment inspiration strikes.

Value Studies

Simplify any reference from 2 to 10 value steps or full grayscale to see the tonal structure of your subject.

Gridlines

Overlay a customizable grid on any reference to nail proportions and perspective.

Color Harmonies

Explore the color wheel with triadic, analogous, split complementary, and monochromatic schemes. Save swatches to your secondary reference panel.

Curated Collections

Ready-to-paint references across landscapes, still lifes, portraits, and more.

Prepare Anywhere, Paint Anywhere

Collect references, plan compositions, and add preparatory sketches on your phone while you’re on the go; then open on your iPad when it’s time to paint.

Designed To Use Offline

Your references go wherever you create. No internet required.

About Artist Reference
Artist Reference is an iPad and iPhone app designed exclusively for traditional artists working with physical mediums like oils, watercolors, charcoal, and pencil. The app provides essential studio tools including value studies, customizable gridlines, color harmonies, and curated reference collections, all in a clean, distraction-free interface that works entirely offline. Artist Reference is developed by an independent developer passionate about supporting the traditional art community.
